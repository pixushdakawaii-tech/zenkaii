# 🌍 Destination Card UI

A simple destination showcase card built using HTML and CSS.

This project displays information about a travel destination using a clean card layout with modern styling and hover effects.

## 🚀 Features

- Responsive card layout
- Hover animations
- Custom shadows
- Clean typography
- Modern UI design

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎯 Learning Goals

This project was created to practice:

- HTML structure
- CSS styling
- Flexbox/Layout techniques
- Hover effects and transitions
- UI design fundamentals

## 🔮 Future Improvements

- Multiple destination cards
- Interactive image gallery
- Dark mode
- Better mobile responsiveness
